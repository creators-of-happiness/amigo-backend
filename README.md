# amigo-backend

## Requirements
- Go 1.24.5+
- Make (optional)
- Docker & Docker Compose (optional)

## Setup
```bash
cp .env.example .env
```

## Quick Start

### Run
```bash
make run
# or
make run-bin
```

### Build
```bash
make build
./bin/app
```

## Docker Compose

```bash
docker compose up --build
```
